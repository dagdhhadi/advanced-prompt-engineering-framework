# Use Case 1: AI Resume Optimization

## Weak Prompt (Before)
Improve my resume.

## Problem
The prompt is vague and does not specify the role, constraints, or output format, leading to generic and low-quality responses.

## Optimized Prompt (After)

### ROLE
You are an AI resume reviewer with 10+ years of experience in technical hiring.

### TASK
Analyze the resume and optimize it for an AI / Prompt Engineering internship role.

### CONTEXT
The candidate has a diploma background and basic AI and Python knowledge.

### CONSTRAINTS
- Do not invent skills or experience
- Keep bullet points concise
- Focus on practical skills and tools

### REASONING STRATEGY
Review each resume section before suggesting improvements.

### OUTPUT FORMAT
1. Resume score out of 100  
2. Key strengths  
3. Key weaknesses  
4. Improved resume bullet points  

## Result
Compared to the weak prompt, the optimized prompt generates structured, role-specific, and actionable resume feedback.
